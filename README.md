# SpringBoot_WebSite

Применяемые технологии: Spring Boot, Spring MVC, Hibernate, Spring Data JPA, PostgreSQL, Tomcat, Git, HTML, применение шаблона MVC.

Разработка серверной части тестового сайта-блога, с возможностью добавлять новые статьи в базу данных сразу на сайте, пееходить на страницы просмотра статей, просмотра информации о блоге, просмотра главной страницы.

Полученные навыки: научился реализовывать сайт на Spring Boot, Spring MVC, работать с ORM и сопутствующими ей фреймворками, применять шаблон проектирования MVC.
